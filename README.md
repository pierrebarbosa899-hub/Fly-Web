# Fly Web 

Oi! Esse é o site que eu fiz com meu grupo pra apresentar nosso TCC do curso de Desenvolvimento Web.

 **Tá no ar aqui:** [pierrebarbosa899-hub.github.io/Fly-Web](https://pierrebarbosa899-hub.github.io/Fly-Web)

## Qual era a ideia?
A gente precisava mostrar nosso projeto de conclusão de curso de um jeito que qualquer pessoa entendesse. Então criamos esse site institucional bem direto ao ponto.

A parte mais legal: o projeto foi **escolhido pra ser apresentado numa palestra na TOTVS**. Fiquei bem orgulhoso disso.

## O que eu usei pra fazer
Usei o básico que funciona bem: **HTML** pra estruturar tudo, **CSS** pra deixar bonito e responsivo, e um pouco de **JavaScript** pro menu do celular funcionar.

Quis treinar bastante Flexbox e Media Queries aqui pra garantir que ficasse bom no computador e no celular.

## O que dá pra fazer no site
- Navegar por todas as seções de boa
- Ver o site certinho no celular, tablet ou PC
- Clicar no menu e ir direto pra parte que interessa

## Minha parte no projeto
Como fizemos em grupo, eu fiquei responsável por todo o front-end. Montei o HTML, fiz o CSS do zero e deixei responsivo. Foi onde mais aprendi na prática.
